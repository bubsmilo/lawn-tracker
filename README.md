# Lawn Tracker

A mobile-friendly lawn fertilizer tracker with:
- Custom lawn areas
- Fertilizer split calculator using 5 kg bags
- Editable application schedule
- Application history with edit/reset
- Settings page with dynamic lawn sizes
- Export/import backup
- PWA manifest, offline support, app icon, and splash assets

## Run locally
Open `index.html` directly, or use a local server:

```bash
python -m http.server 5173
```

Then open `http://localhost:5173`.

## Deploy to Vercel
Upload this folder to GitHub, then import the repo into Vercel as a static site.
No build command is required.
